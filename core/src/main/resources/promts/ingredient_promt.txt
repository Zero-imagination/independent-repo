Для блюда салат Цезарь напиши все ингредиенты необходимые для его приготовления в следующем формате:
('название ингредиента', цена, 'описание продукта', количество, 'дата истечения срока годности', null, null),
при этом есть условия: название ингредиента на русском языке, цена это цело число за килограмм, описание продукта это краткое описание в одно предложение, количество это целое число грамм для приготовления одной порции, дата истечения срока годности это дата в формате yyyy-mm-dd в месячном диапазоне от сегодняшней даты.
пример для блюда Борщ что должно получиться:
('Свекла', 200, 'Свекла обыкновенная', 30, '2024-02-28', null, null),
('Картофель', 70, 'Картофель', 20, '2024-02-20', null, null),
('Говядина', 550, 'Говяжья вырезка', 35, '2024-02-21', null, null),
пример в ответе выводить не нужно
